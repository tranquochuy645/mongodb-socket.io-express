API GUIDE

Create new user

HTTP request to "https://database-app.onrender.com/api/create_user/"

method: 'POST', headers: { 'Content-Type': 'application/json' }, body: { 'username': username, 'password': password }

~$ curl -X POST -H "Content-Type: application/json" -d '{"username":"test_user","password":"test_password"}' https://database-app.onrender.com/api/create_user/ -w "\nResponse Code: %{http_code}\n"

Login/Get your token

HTTP request to "https://database-app.onrender.com/api/login/"

method: 'POST', headers: { 'Content-Type': 'application/json' }, body: { 'username': username, 'password': password }

~$ curl -X POST -H "Content-Type: application/json" -d '{"username":"test_user","password":"test_password"}' https://database-app.onrender.com/api/login/ -w "\nResponse Code: %{http_code}\n"

Get your data

HTTP request to "https://database-app.onrender.com/api/get_data/"

method: 'GET', headers: { 'Content-Type': 'application/json', 'token': yourToken }

~$ curl -X GET -H "Content-Type: application/json" -H "token: 642135fad6d53079497933c3" https://database-app.onrender.com/api/get_data/ -w "\nResponse Code: %{http_code}"

Get your device's socketId

HTTP request to "https://database-app.onrender.com/api/get_devices/"

method: 'GET', headers: { 'Content-Type': 'application/json', 'token': token }
